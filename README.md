# 42.ft_ls
Reimplementation of 'ls' to learn more about the filesystem

ls sort by default the file in an alphabetical order

Some flags need to be implemented:
	-a	OK
	-l
	-R
	-r	OK
	-t


There are the allowed functions:
◦ write
◦ opendir
◦ readdir
◦ closedir
◦ stat
◦ lstat
◦ getpwuid
◦ getgrgid
◦ listxattr
◦ getxattr
◦ time
◦ ctime
◦ readlink
◦ malloc
◦ free
◦ perror
◦ strerror
◦ exit
