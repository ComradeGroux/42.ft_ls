# 42.ft_ls
Reimplementation of 'ls' to learn more about the filesystem

Some flags need to be implemented:
	-a
	-l
	-R
	-r
	-t


There are the allowed functions:
◦ write
◦ opendir
◦ readdir
◦ closedir
◦ stat
◦ lstat
◦ getpwuid
◦ getgrgid
◦ listxattr
◦ getxattr
◦ time
◦ ctime
◦ readlink
◦ malloc
◦ free
◦ perror
◦ strerror
◦ exit
